# Phase 7: Complete Subclass Catalog

Phase 7 completes the Legacy 2014 subclass data used by Homebrew God.

## Delivered

- 118 subclasses with completed summaries and descriptions
- 623 subclass features across every required class level
- Structured choices, resources, action economy, effects, save DCs, and class-level scaling
- Expanded spell lists and spell grants for eligible subclasses
- Eldritch Knight and Arcane Trickster third-caster progression and school restrictions
- Cleric domain spells and Channel Divinity options
- Paladin oath spells and Channel Divinity options
- Complete Artificer specialist, Warlock patron, and Sorcerous Origin records
- Shared, non-stacking Extra Attack handling for subclass variants
- SRD subclasses prioritized first in their class lists

## Validation

The strict subclass validator rejects placeholder text and missing required feature levels. The browser self-test suite adds 26 Phase 7 regressions covering the catalog and its class-specific mechanics.

Subclass schema version: 2

Character schema version: 9

Spell references that are not yet part of the main spell catalog include complete inline fallback data so subclass grants remain usable. Expanding the main spell catalog remains Phase 15 work.

Phase 8 is not included in this release.
